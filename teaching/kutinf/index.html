A Kutatómunka információs eszközei

A Kutatómunka információs eszközei nevű kurzus során megismerkedhettek olyan információs eszközökkel, amelyek segíteni fogják a kutatómunkát. Bővebb információt a hivatalos honlapon találsz.

A megtartott 2-3 órám során megismerkedhettek alap weblapszerkesztéssel (HTML, CSS, Javascript), illetve a Jupyter Notebook egyik hasznos ágával, a Slideshow készítéssel. Egy beadandó lesz, ami lényegében az órán elmondottak (alább fel lesznek rakva a dolgok hozzá) reprodukciója.

Legutóbbi frissítés: 20/02/13.

Lectures:

  1. Bevezetés
  2. Basic HTML webpage
  3. Basic CSS
  4. Basic Javascript
  5. Basic Ipyhton Slideshow
